## 📖 專案簡介
本工具包協助銀行業落實雇主防治義務（職安法 §22-2）、建立健全的申訴調查處理機制、強化稽核與法令遵循、推動教育訓練與全員宣導。

## 🎯 適用對象
| 對象 | 主要使用文件 |
|---|---|
| 稽核處、法遵處 | 01 自行查核 + 02 法規架構圖 |
| 人力資源處 | 05 教育訓練簡報 |
| 申訴處理單位、調查小組 | 01 個案處理檢核  |
| 各級主管 | 04 主管版懶人包 + 05 教育訓練簡報 |
| 全體員工 | 03 員工版懶人包 |
| 教育訓練講師 | 05 教育訓練簡報 |

## 📦 檔案清單
| # | 檔名 | 格式 | 用途 |
|---|---|---|---|
| 01 | 自行查核工作底稿_v2 | xlsx | 42 項自行查核 + 42 步驟個案檢核（6 分頁）|
| 02 | 查核流程與法規架構 | drawio | 法規體系圖 + 查核流程圖 |
| 03 | 員工版懶人包 | docx | 6 大區塊宣導 |
| 04 | 主管版懶人包 | docx | 8 大區塊主管責任 |
| 05 | 教育訓練簡報 | pptx | 30 張投影片、3 小時課程 |

所有檔案位於 `docs/` 資料夾。

## 🚀 快速開始
- **自行查核** → 開啟 `docs/01_*` + `docs/02_*`
- **宣導訓練** → 列印 `docs/03_*` `docs/04_*`、播放 `docs/05_*`

## 🏛️ 法規依據
- 職業安全衛生法 §22-1 ~ §22-5（114.12.19 修正，115.7.1 施行）
- 職場霸凌防治措施準則（115.6.23）
- 地方主管機關受理最高負責人職場霸凌事件申訴處理辦法
- 職場霸凌調查專業人士培訓及專業人才庫建置要點
- 職場霸凌防治措施指導手冊（勞動部 115.6）
