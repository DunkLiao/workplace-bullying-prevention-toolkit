## 2. 職場霸凌定義與認定原則

```mermaid
mindmap
  root((職場霸凌認定))

      持續以冒犯、威脅、冷落、孤立、侮辱或其他不當言行

      情節重大者

    綜合審酌
      事件背景
      發生頻率

      行為人動機
      行為目的
    常見樣態
      刻意排擠或冷落
      阻礙工作或隱瞞資訊
      權力欺壓或不合理目標
```

## 5. 知悉事件後的立即有效措施

```mermaid
mindmap
  root((知悉後立即有效措施))
    因申訴而知悉
      考量申訴人意願
      採取隔離等避免再發措施
      不得不利對待
      提供或轉介協助
      啟動調查程序
      申訴人有意願時可協調
      依結果懲戒或處理
    非因申訴而知悉
      訪談相關人員
      釐清與查證事實
      告知權益與救濟途徑
      依意願協助協調或申訴
      適度調整工作內容或場所
      提供法律、醫療、心理或社福資源
    即使無申訴意願
      仍應採取適當措施
      留存處理紀錄
```

## 7. 調查、協調與證據處理

```mermaid
mindmap
  root((調查與協調程序))
    調查原則
      客觀
      公正
      公平
      保護隱私
      不公開處理
    協調程序
      以申訴人意願為前提
      協調人員須經當事人同意
      任一方無意願即停止
      逾一個月未有共識即停止
      達成共識應作成紀錄
      不成立時續行調查
    調查方式
      訪談當事人
      訪談相關人員
      查驗物證與文件
      必要時外部專業協助
    大型事業單位
      一百人以上
        受理申訴後十五個工作日內組成調查小組
        外聘成員比例不得少於二分之一
```

## 10. 最高負責人職場霸凌事件外部申訴

```mermaid
  root((最高負責人事件))
    適用情境

    最高負責人認定

      董事長或理事主席

      實質控制人事、財務或業務者
      持股二十%以上等具體事證情形
    申訴文件
      書面提出
      申訴人基本資料

      事實內容與證據
      申訴年月日
      十個工作日內補正
    地方主管機關程序
      十個工作日內通知是否受理
      受理後十個工作日內開始調查
      二個月內完成調查
      必要時延長一個月
      調查報告完成後一個月內作成決定
    調查保障
      調查成員至少二人
      任一性別比例不得少於三分之一

      不公開調查
      提供資源轉介或法律諮詢
```
